Robot Creation


Random Robot Generation

By creating many random robots from a list of 3-Dimensional shapes and sizes our program will be able to find odd solutions to tasks one might not think of normally.


Efficient

The creation of a robot takes less than a second allowing use to create many within a short period of time. Each generation of bots will be created before the user's eyes in no time at all.

Terrain Generation

No Need To Create Your Own Terrain

There will be no need to waste time creating terrain for the bots to traverse. Our software will offer a unique number of preset maps that will be challenging as well has help to provide the best results.

Random Terrain Generation

Eventually we wish to add the ability for the software to create random terrain on the fly. Users will select which environmental traits they would like the bots to work under and a randomly created map will be made from the ground up. This feature is not currently in development but may be apart of the final release.
