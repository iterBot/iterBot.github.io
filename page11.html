  • Map Creation

A better form of uploading terrain has been added to help allow more variety. A member of the iterBot team can now create a map quickly following certain size restraints and directly add the map to the software. The program itself will add the start and finish point from which fitness is scored as well as duplicate the map for each bot to use. iterBot will have a good amount of maps at completion but the ability to add more is there in the future.
