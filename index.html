Welcome to our site!

Our senior project is about creating random land traversing robots using genetic algorithms within the 3D engine called Unity. 

Be sure to check out the weekly updates to stay updated on our project.

  1. Random Robot Generation - Adding randomization to the creation of robots adds a whole new depth to the project. We will be able to create and test non-trivial designs that might perform well under certain environments that most people wouldn't think of. For example a robot with 3 sections differing in weight in size might be able to fling itself across the terrain successfully. 
  2. Genetic Algorithms - Genetic algorithms are what allow this whole project to work. By scoring each bot with a fitness which determines how successful they were we can then use their structure to seed a whole new generation of bots that have similar features to the top performers. The easiest way to think about genetic algorithms is to imagine it is a simulated version of natural selection where we only allow the best robots to seed the further generations.
  3. Unity - Unity is a very well known 3D engine commonly used to create video games and life like cinematics. It has the ability to simulate physics which we will be using for the purposes of our project to help us test the robots we generate.
